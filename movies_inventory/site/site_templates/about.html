About



Creator: Tarun Patel

This site was created to showcase using Flask to call a 3rd party API from the TMDB REST API. The Home page shows 10 movies from the popular, upcoming, and now playing endpoints. API is called using arguments passed through to functions.

The search API takes in user input to pull the first 5 results from the movie search endpoint. If user is logged in they can choose to favorite the movie and save it to their profile. These will be tracked by different logins.
